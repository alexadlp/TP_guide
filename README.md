# Projet AlgoBox Collaboratif

## Description
Ce projet permet à plusieurs étudiants de collaborer sur des fichiers AlgoBox (.algo).  
Il contient des exemples d’algorithmes simples : somme, moyenne, factorielle, etc.

## Contenu du projet
- `somme.algo` : calcule la somme de deux nombres
- `moyenne.algo` : calcule la moyenne de trois nombres
- `factorielle.algo` : calcule la factorielle d’un nombre

## Prérequis
- Git installé : https://git-scm.com
- AlgoBox installé pour ouvrir et modifier les fichiers `.algo`
- Compte GitHub pour collaborer
